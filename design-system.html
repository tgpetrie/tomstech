Design System

All tokens, components, and patterns from the production website. Import design-tokens.json into Figma via Token Studio, then use this page as your component reference.

How to import into Figma

  1. 1
    In Figma, open Plugins → Token Studio for Figma (install free from Community if needed)
  2. 2
    In Token Studio, click Load from file/URL → select design-tokens.json from this project folder
  3. 3
    Click Apply to document — this creates all color, typography, spacing, shadow, and radius styles
  4. 4
    For native Figma Variables: Token Studio → Sync → Figma Variables (requires Token Studio Pro) — or manually create variables from the token values listed below
  5. 5
    Build components by referencing the token names (e.g. primitive/color/ink, primitive/shadow/flat-md)
Fonts
Four Google Fonts — all variable weight (except IBM Plex Mono). Raleway is the H1/hero face; Bricolage is body/cards with optical size axis; Space Grotesk covers UI headings H2+. Add all to your Figma file via the text panel.
