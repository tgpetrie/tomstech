Design System

All tokens, components, and patterns from the production website. Use the Theme Preview selector to see all 11 themes live. Import design-tokens.json into Figma via Token Studio, then use this page as your component reference.

How to use this system

Tip: Token Studio now requires a paid plan. Use the free community plugin json-to-figma-styles instead — it creates native Figma Color Styles and Text Styles that work on the free Figma plan.

  1. 1
    Add <link rel="stylesheet" href="tomstech-system.css"> to your HTML <head>
  2. 2
    Add data-theme="earthy" to your <html> tag to activate a theme
  3. 3
    Use the class names shown on this page — all components render from that one CSS file
  4. 4
    For Figma: open Plugins → Community, install "json-to-figma-styles", load design-tokens.json
  5. 5
    Build components by referencing token names, e.g. primitive/color/ink, primitive/shadow/flat-md
Fonts
Raleway is the sole typeface (v2.0 single-font system). Its variable axis (100–900) creates all hierarchy: 800 for hero H1, 700 for section H2 / card H3 / labels / buttons / logo, 400 for body copy. Label and badge roles use Raleway 700 + text-transform: uppercase + letter-spacing.
